## C++ homework 
#### Dynamic Memory Allocation
1. Make sure it is **NULL** before allocate a pointer.
2. Make sure it is **not NULL** before delete a allocated pointer.
3. Make sure it is **not NULL** after allocate a pointer.
   ( For some reason like memory overload, sometimes it would not throw error and make the pointer NULL )
4. delete a pointer should be as possible as close to new a pointer.
